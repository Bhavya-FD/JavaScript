Javascript Data Types

JavaScript provides different data types to hold different types of values.
There are two types of data types in JavaScript.

  1. Primitive data type
  2. Non-Primitive data type

A. Premitive Data Type

  1. Number
  2. Null
  3. String
  4. Symbol
  5. BigInt
  6. Boolean
  7. Undefined

1.Number

represents numeric values e.g. 100
The number type represents both integer and floating point numbers.

2.Null

represents null i.e. no value at all

3.String

represents sequence of characters e.g. "hello"
In JavaScript, there are 3 types of quotes.

Double quotes: "Hello".

Single quotes: 'Hello'.

Backticks: `Hello`.

4.Symbol

represent symbols

5.BigInt

represents big integers/numbers & ends with n

6.Boolean

represents boolean value i.e. either false or true

7.Undefined

represents undefined value(which is declared but not assigned any value)

B. Non-Premitive Data Type

  1. Array
  2. Object

1.Array

JavaScript array is an object that represents a collection of similar type of elements.

The syntax of creating array using array literal is given below:
var arrayname=[value1,value2.....valueN];

2.Object

A javaScript object is an entity having state and behavior (properties and method). For example: car, pen, bike, chair, glass, keyboard, monitor etc.

object={property1:value1,property2:value2.....propertyN:valueN}

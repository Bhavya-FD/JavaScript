variables in JS

Variables:

A variable is a “named storage” for data. We can use variables to store goodies,
visitors, and other data. To create a variable in JavaScript, use the let keyword.

We have two types of variables in JS and one constant

  1. var
  2. let
  3. const

1. var

var a;

var a = 10; // Initialization and Declaration or Assignemnt

var a = 15; // Re-Declaration is possible

a = 25; // Re-Assignment is possible

console.log(a);

output: a = 25

2. let

let _a= 90 // Initialization and Declaration or Assignemnt

let _a = 40 // Re-Declaration is not possible

_a= 75 // Re-Assignemnt is Possible

console.log();

output: _a = 40

3.Constants (const):

To declare a constant (unchanging) variable, use const instead of let:

EXAMPLE:const myBirthday = '18.04.1982';

myBirthday = '01.01.2001'; // error, can't reassign the constant!

const COLOR_RED = "#F00";
const COLOR_ORANGE = "#FF7F00";
// ...when we need to pick a color
let color = COLOR_ORANGE;
alert(color); // #FF7F00

const fav_color = "Black"; // Initialization and Declaration or Assignemnt

const fav_color = "Red" // Re-Declaration is not possible

fav_color = "Yellow" // Re-Assignemnt is not possible

console.log(fav_color);

output: fav_color = Black

There are some rules while declaring a JavaScript variable (also known as identifiers).

  • Name must start with a letter (a to z or A to Z), underscore( _ ), or dollar( $ ) sign.
  • After first letter we can use digits (0 to 9), for example value1.
  • JavaScript variables are case sensitive, for example x and X are different variables.
  • There is a list of reserved words, which cannot be used as variable names because they are used by the language itself.
    For example: let, class, return, and function are reserved.
 
    Correct JavaScript variables: 
    var x = 10;  
    var _value="bhavya"; 
    
    Incorrect JavaScript variables:
    var  123=30;  
    var *aa=320;  

    Example of JavaScript variable: 
    var x = 10;  
    var y = 20;  
    var z=x+y;  
    document.write(z);

Use Strict

we need to define a variable before using it. But in the old times, it was technically possible to create a variable by a mere assignment of the value without using let. This still works now if we don’t put "use strict" in our scripts to maintain compatibility with old scripts.

    num = 5; 
    alert(num);

the variable "num" is created if it didn't exist

    "use strict";
    num = 5;
    alert(num);

It will show 'error: num is not defined'

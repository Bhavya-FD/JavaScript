Introduction to JS

What is JS

JavaScript was initially created to “make web pages alive”.

JavaScript (js) is a light-weight object-oriented programming language which is used by several websites for scripting the webpages. It is an interpreted,
full-fledged programming language that enables dynamic interactivity on websites when applied to an HTML document.

Features of JavaScript

There are following features of JavaScript:

  • It is a case-sensitive language.
  • It is a light-weighted and interpreted language.
  • JavaScript follows the syntax and structure of the C programming language. Thus, it is a structured programming language.
  • JavaScript is supportable in several operating systems including, Windows, macOS, etc.
  • JavaScript is an object-oriented programming language that uses prototypes rather than using classes for inheritance.

Ways to Apply

Javascript example is easy to code. JavaScript provides 3 places to put the
JavaScript code: within body tag, within head tag and external JavaScript file.

Comments

Comments are written using double slash(//) in JS.

Types of JavaScript Comments

There are two types of comments in JavaScript:

  1. Single-line Comment
  2. Multi-line Comment

1. JavaScript Single line Comment

It is represented by double forward slashes (//). It can be used before and after the statement.

// It is single line comment

2. JavaScript Multi line Comment

It can be used to add single as well as multi line comments. So, it is more convenient.
It is represented by forward slash with asterisk then asterisk with forward slash.

/* It is multi line comment.*/
